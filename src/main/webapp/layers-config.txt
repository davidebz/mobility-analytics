/*
   id: sigla del layer. parola univoca e senza spazi
   icons: icona del layer e range per visualizzare le bollicine dello stato.
   stationType: station type dei dati del layer
   format: il formato dei dati forniti dall'url. Valori implementati: integreen, wms
   projection: projezione dei dati gis. usare il nome riconosciuto da openlayer. alcune projezioni richiedono l'inclusione di ulteriori liberie a supporto
   custom-function: ulteriore personalizzazione del comportamento standard mediante una funzione programmata ad hoc
   color: colore in cui viene visualizzato il marker del layer
   mainData: data types che vengono visualizzati nella section del dettaglio come default
 */
